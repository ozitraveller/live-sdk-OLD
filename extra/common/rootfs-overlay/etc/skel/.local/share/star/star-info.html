Star

Star is a lightweight distro based on Devuan Stable. The whole philosophy of Star is "keep it stable and light",
so there is a minimal number of default applications.

If you like to customise your desktop, then this is the distro you you!

Features

  • Built on Devuan stable
  • Xfce, fluxbox, openbox, jwm, i3, netinst, ...
  • Small selection of applications, including: web browser, text editor, image viewer, pdf viewer, file manager, terminal, media player
  • Backports enabled by default
  • Both text and Graphical installers
